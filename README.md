Program Description
-------------------
This Student Revision Planner is a command-line application that helps students organise, track, and prioritise revision tasks.
Users can add, delete, search, mark as complete, and sort tasks by priority or deadline.
All task data is saved to and loaded from a JSON file for persistent storage.
The program includes robust error handling, clear user interface, modular functions, classes, search and sorting algorithms,
and file operations to meet all IY499 assessment requirements.
